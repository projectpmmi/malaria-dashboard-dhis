# DHIS2 app example
This example DHIS2 app allows you to change an organisation unit's code.

## Install
```bash
npm install
# or
yarn install
```

## Develop
```bash
npm start
# or
yarn start
```

## Build
```bash
npm run build
# or
yarn build
```
